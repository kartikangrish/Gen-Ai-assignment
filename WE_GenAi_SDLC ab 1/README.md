# WE_GenAi_SDLC ab 1

This project is a small Python application created for the GitHub Copilot lab exercise. It demonstrates creating variables, functions, comment-driven code generation, inline chat style additions, and chat-assisted method creation.

## Files

- `main.py` - runnable Python application
- `LAB_EVIDENCE.md` - step-by-step lab evidence and screenshot checklist
- `.vscode/extensions.json` - recommended VS Code extensions

## What The App Does

The application:

1. Stores a user's name, email, and location.
2. Creates a greeting message using those details.
3. Calculates total, average, highest, and lowest scores from five subject marks.
4. Combines the profile and score details into one final display.

## Run The Application

From this folder, run:

```powershell
python main.py
```

Expected output:

```text
Hello Kartik! Your email is kartik@example.com, and you are from Bengaluru.
Subject Scores: 88, 92, 79, 95, 84
Total Score: 438
Average Score: 87.60
Highest Score: 95
Lowest Score: 79
```
