# Calculator Lab 3

A basic calculator web application built with HTML, CSS, and JavaScript.

## Features

- Addition, subtraction, multiplication, and division
- Clear and delete buttons
- Decimal number support
- Division-by-zero message
- Responsive calculator UI

## Run

Open `index.html` in a browser, or run a local static server:

```powershell
Set-Location .\calculator-lab-3; python -m http.server 5500
```

Then open:

```text
http://localhost:5500
```

## Use

Click the number and operator buttons to build a calculation. Press `=` to show the result, `C` to clear the display, and `DEL` to remove the last character. The calculator supports decimal values and shows a clear message when division by zero is attempted.
